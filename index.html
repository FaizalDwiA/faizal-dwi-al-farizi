Fullstack Developer

CV.Rosin

2024-2025

Di CV. Rosin saya berperan sebagai fullstack developer dengan mengembangkan sistem absensi berbasis web yang terintegrasi face detection dan geolokasi. Proyek ini menggunakan Node.js, CodeIgniter 3, dan MySQL sebagai database, serta dideploy melalui FileZilla dengan server Node.js untuk proses pencocokan wajah. Sistem mencakup menu inti seperti Absensi, History, Pendaftaran, Izin Karyawan, Lokasi, Tabulasi, hingga Rekap Cuti Web.

  • Merancang dan membangun aplikasi absensi berbasis web dengan fitur deteksi wajah dan lokasi secara akurat.
  • Mengembangkan sisi front-end dan back-end menggunakan kombinasi Node.js dan CI3, dengan integrasi database MySQL yang terstruktur.
  • Membuat server Node.js khusus untuk proses face matching karyawan.
  • Mengelola proses deployment aplikasi melalui FileZilla serta memastikan sistem berjalan stabil di server produksi.
  • Mengimplementasikan modul-modul utama, termasuk: Absensi, History Absensi, Daftar Pendaftaran, Group, Izin Karyawan, Jenis Izin, Libur, Log, Lokasi, Tabulasi, dan Rekap Cuti Web.
